# Decision Intelligence Console — Master Package (Updated)

This package consolidates the current direction for the internal **Decision Intelligence Console** and its relationship to the external product **Shortlist**.

## What changed in this update
- Clarifies that **Shortlist** is the sellable external product.
- Clarifies that the **Console** is the internal operating system for building, managing, and improving products, intelligence workflows, and live applications.
- Updates the architecture to support:
  - suggestion engine
  - prediction engine
  - API intelligence mapping
  - marketing/design machine
  - self-improving agent systems
  - shared infrastructure strategy
  - live app operations

## Core idea
The internal console should become a **decision intelligence command center** for:
- finding opportunities
- evaluating markets and vendors
- orchestrating agents
- monitoring experiments and production systems
- managing source intelligence
- accelerating product creation
- launching and operating niche applications

Shortlist remains the first clear commercial wedge.
